# Test Case Generator UI

Modern React frontend for the Test Case Generator microservice powered by Claude Agent SDK.

## Features

- Modern UI with gradient backgrounds
- Dual Mode: Manual input or JIRA integration
- Real-time test case generation
- Export as JSON
- Responsive design

## Setup

```bash
npm install
cp .env.example .env.local
npm run dev
```

## Deployment

Deploy to Railway, Vercel, or Netlify. See DEPLOYMENT.md in backend for details.
